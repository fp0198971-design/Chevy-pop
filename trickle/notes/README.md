# SneakerPro - Landing Page

## Descripción
Landing page moderna y minimalista para una marca de tenis deportivos premium. Diseñada con tema oscuro, efectos de glassmorphism y animaciones suaves.

## Características Principales
- **Navbar fija** con efecto glass y menú responsive tipo hamburguesa
- **Hero carousel** con 3 imágenes destacadas de productos
- **Galería de productos** con tarjetas interactivas
- **Sidebar de detalles** para cada producto con selección de color y talla
- **Botón CTA de Instagram** con estilo degradado
- **Footer** organizado con información de marca, redes sociales y contacto

## Tecnologías
- HTML5 semántico
- Bootstrap 5.3
- React 18
- AOS (Animate On Scroll)
- Lucide Icons
- Google Fonts (Poppins)

## Paleta de Colores
- Fondo oscuro: `#0a0a0a`
- Fondo secundario: `#1a1a1a`
- Color de acento: `#00ff88` (verde lima)
- Texto claro: `#f5f5f5`
